# EasyCollect

A payment collection platform designed for Nigerian students to easily collect money for handouts, course dues, parties, and projects.

## Features

- **Host Registration Required**: Only registered hosts can create collection links
- **No Registration for Contributors**: Contributors can pay without creating accounts
- **Custom Fields**: Hosts can create custom fields for contributors to fill (Student ID, Department, etc.)
- **Payment Verification**: Bank transfer payments with countdown timer and verification
- **Transaction Receipts**: Detailed receipts after successful payments
- **Share Options**: Share via WhatsApp, Email, or SMS

## Tech Stack

- React + TypeScript
- Tailwind CSS
- React Router
- Lucide React Icons

## Deployment

### Deploy to Vercel

1. Install Vercel CLI:
```bash
npm i -g vercel
```

2. Deploy:
```bash
vercel --prod
```

Or connect your GitHub repository to Vercel for automatic deployments.
